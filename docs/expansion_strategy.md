## 전략 1: "글씨인 키트" — 소량 실물 상품 판매 (가장 즉시 실행 가능)

### 왜 이것부터인가
- 이미 수업에서 사용하는 재료(붓펜, 먹, 화선지 등)에 대한 큐레이션 노하우가 있음
- 수강생이 "집에서도 연습하고 싶다"는 수요가 존재 (원데이 클래스 참여자 특히)
- 웹사이트에 이미 관리자 시스템이 있어 상품 관리 기능 확장이 용이

### 구체적 첫 단계

**Phase A: 검증 (1~2주)**
1. 인스타그램 스토리에 "집에서 캘리 연습할 때 어떤 도구 쓰시나요?" 투표 진행
2. 기존 수강생 10명에게 DM으로 "글씨인이 추천 도구 세트를 만들면 구매 의향이 있는지" 간단 설문
3. 가격대 테스트: 2만원대 / 3만5천원대 / 5만원대 중 어디에 반응이 오는지

**Phase B: MVP 제작 (2~3주)**
```
[글씨인 스타터 키트] 구성 예시 (예상 원가 8,000원 / 판매가 25,000원)
├── 글씨인 추천 붓펜 2종 (쿠레타케 ZIG + 펜텔 아쿠아시)
├── 연습용 반숙지 20매 (A4 커팅, 글씨인 로고 워터마크)
├── 글씨인 오리지널 연습 가이드 엽서 5매 (QR코드 → 영상 연결)
├── 글씨인 스티커 1매
└── 대표 작가 손글씨 메시지 카드 1매
```

**Phase C: 판매 채널 (즉시)**
- **1차:** 수업 현장 판매 (원데이 클래스 종료 후 "집에서도 해보세요")
- **2차:** 인스타그램 DM 주문 → 네이버 스마트스토어 개설 (수수료 5.5%)
- **3차:** 웹사이트 `/shop` 페이지 추가 (Supabase에 products 테이블 추가)

**예상 수익 시뮬레이션:**
- 월 30세트 판매 시: 매출 75만원, 순이익 약 51만원
- 원데이 클래스(월 4회 × 평균 5명) 참여자 전환율 30% = 6세트/월 (최소 보장)

### 확장 경로
- 시즌 키트 (설날 족자 키트, 어버이날 카드 키트, 크리스마스 엽서 키트)
- "글씨인 프리미엄 키트" (전문가용 붓 + 먹물 + 화선지 세트, 7만원대)

## 전략 2: "마이크로 콘텐츠" 전략 — 숏폼 영상 기반 유입 확대

### 왜 이것인가
- 캘리그라피는 "글씨 쓰는 과정" 자체가 ASMR 수준의 시청 만족도를 제공하는 콘텐츠
- 릴스/숏츠 1개 제작에 5분도 안 걸림 (실제 글씨 쓰는 장면 촬영 → 2배속 편집)
- 알고리즘이 지역 타겟팅을 해줌 → 김포/일산/파주 거주자 자연 유입

### 구체적 첫 단계

**매주 3개 콘텐츠 공식 (촬영 총 30분 이하):**

| 요일 | 콘텐츠 유형 | 예시 | 촬영 시간 |
|------|-----------|------|----------|
| 화 | 글씨 ASMR (15초) | 오늘의 한 문장을 붓으로 쓰는 탑뷰 영상 | 5분 |
| 목 | Before/After (30초) | 수강생 첫 수업 vs 3개월 후 비교 | 10분 |
| 토 | 수업 비하인드 (20초) | 원데이 클래스 현장 스케치 | 10분 |

**촬영 세팅 (초기 투자 약 5만원):**
```
필수 장비:
├── 스마트폰 거치대 (탑뷰 촬영용, 쿠팡 15,000원)
├── 조명: 자연광 or 링라이트 (이미 보유 가정)
└── 편집: CapCut 무료 앱 (2배속 + 자막 + BGM)

촬영 포인트:
├── 배경: 깨끗한 목재 책상 or 화선지 깔기
├── 손목까지만 노출 (얼굴 노출 부담 제거)
└── 마지막 컷에 "글씨인아트센터" 로고 삽입
```

**해시태그 전략:**
```
고정 태그: #글씨인아트센터 #김포캘리 #캘리그라피 #손글씨
지역 태그: #김포 #김포맘 #김포취미 #일산 #파주
감성 태그: #글씨스타그램 #캘리일상 #붓글씨 #먹놀이
```

**KPI (3개월 목표):**
- 팔로워: 현재 → +500명
- 릴스 평균 조회: 1,000뷰 이상
- DM 문의: 주 3건 이상
- 원데이 클래스 예약 중 "인스타 보고 왔어요" 비율 20% 이상

## 전략 3: "글씨인 B2B" — 지역 소상공인 대상 캘리그라피 서비스

### 왜 이것인가
- 김포는 신도시 개발로 소규모 카페, 베이커리, 공방이 급증 중
- 이들의 공통 니즈: "감성적인 간판/메뉴판/POP를 원하지만 어디에 의뢰할지 모름"
- 대표 작가의 국전 특선 경력은 "프리미엄 캘리그라피"로 포지셔닝 가능
- 단가가 높고 (건당 10~50만원), 포트폴리오가 곧 홍보 자산이 됨

### 구체적 첫 단계

**Phase A: 서비스 메뉴 정의**

| 서비스 | 납품물 | 예상 단가 | 제작 소요 |
|--------|--------|----------|----------|
| 메뉴판 캘리 | 손글씨 메뉴 원본 + 고해상도 스캔 파일 | 15~25만원 | 2~3일 |
| 간판 글씨 | 캘리그라피 원본 + 벡터 변환 파일(ai) | 30~50만원 | 3~5일 |
| 시즌 POP | 매장 내 시즌 홍보물 (A3~A2) | 5~10만원/장 | 1일 |
| 패키지 디자인 | 쇼핑백, 포장지 등 글씨 디자인 | 20~40만원 | 3~5일 |

**Phase B: 첫 고객 확보 (이것이 핵심)**

1. **김포 한강신도시 카페 5곳에 "무료 체험 제안"**
   - "메뉴판 캘리그라피 1장을 무료로 제작해 드립니다. 대신 인스타에 태그 1회 부탁드려요."
   - 직접 방문하여 메뉴판 스타일 상담 → 3일 내 납품
   - 이 5건이 포트폴리오 + 입소문의 시작점

2. **납품 후 콘텐츠화**
   - "OOO 카페 메뉴판을 써드렸습니다" 인스타 게시
   - 카페 사장님이 자발적으로 공유하는 2차 노출 효과
   - 웹사이트 `/portfolio` 또는 `/works` 페이지에 B2B 포트폴리오 섹션 추가

3. **김포시 소상공인 지원사업 연계**
   - 김포시 소상공인지원센터의 "간판개선사업", "상권활성화사업" 등과 연계
   - 소상공인이 지원금으로 캘리그라피 서비스를 의뢰하는 구조
   - 관할: 김포시청 일자리경제과 / 소상공인시장진흥공단 경기서부센터

**예상 수익:**
- 월 2~3건 수주 시: 월 30~75만원 추가 수익
- 6개월 후 단가 정상화 시: 월 50~150만원

## 전략 4: "수강생 커뮤니티 → 전시 → 브랜드화" 순환 구조

### 왜 이것인가
- 이미 "글씨인 커뮤니티" 운영 중 (대표가 운영작가)
- 수강생이 "작품을 발표할 장"이 없으면 중급 이후 이탈률이 높아짐
- 소규모 전시는 비용 대비 브랜드 임팩트가 매우 큼

### 구체적 첫 단계

**"글씨인展" 미니 전시 기획 (연 2회)**

```
전시 규모: 수강생 10~15명 참여, 작품 30~50점
장소 옵션:
├── [무료] 김포아트빌리지 갤러리 (김포문화재단 운영, 무료 대관 가능)
├── [무료] 김포시립도서관 전시공간
├── [저비용] 한강신도시 내 카페 갤러리 대관 (10~20만원)
└── [자체] 글씨인아트센터 내부 (수업 시간 외 활용)

비용 구조 (카페 갤러리 기준):
├── 대관료: 15만원
├── 액자/마운팅: 참여 수강생 자비 (1인 2~3만원)
├── 리플렛 인쇄: 5만원 (100부)
├── 다과: 5만원
└── 총 비용: ~25만원 (참여비 수강생 분담 시 실질 부담 5만원 이하)
```

**이 전시가 만드는 가치:**
1. **수강생 동기부여:** "내 작품이 전시된다" → 수업 지속률 상승
2. **입소문 마케팅:** 수강생 가족/친구 방문 → 자연 홍보 (1인당 평균 3~5명 초대)
3. **콘텐츠 생산:** 전시 현장 사진/영상 → SNS 콘텐츠 1~2주치
4. **언론 노출:** 김포신문, 경기일보 등 지역 언론에 보도자료 배포 (무료)
5. **포트폴리오:** "글씨인아트센터는 전시까지 하는 곳"이라는 브랜드 격 상승

**보도자료 배포처:**
```
├── 김포신문 (gimponews.co.kr) - 기사 게재 확률 높음
├── 김포타임스
├── 경기일보 - 김포지사
├── 지역 커뮤니티: 김포맘카페, 김포시민모임 등
└── 김포시청 문화체육과 → 시정소식지 게재 요청
```

## 전략 5: "디지털 콘텐츠 판매" — 연습 가이드 PDF/영상

### 왜 이것인가
- 한 번 만들면 반복 비용 없이 계속 판매 가능 (한계비용 ≈ 0)
- 원데이 클래스에 올 수 없는 비김포 거주자도 고객으로 전환
- 웹사이트의 회원 시스템(Supabase Auth + RLS)이 이미 구축되어 있어 유료 콘텐츠 접근 제어 가능

### 구체적 첫 단계

**상품 1: "글씨인 연습 가이드 PDF" (9,900원)**
```
구성:
├── 기본 획순 가이드 (10페이지)
├── 일상 문구 30선 (생일, 감사, 위로 등 실용 문장)
├── 따라쓰기 연습지 (인쇄하여 사용, 20매)
├── 도구 추천 가이드 (초보자용)
└── QR코드 → 시범 영상 3개 (유튜브 비공개 링크)

제작 소요: 7~10일 (기존 수업 자료 재편집)
디자인: Canva Pro (월 13,000원) or 미리캔버스 (무료)
```

**상품 2: "영상 레슨 팩" (29,900원)**
```
구성:
├── 기초 캘리그라피 영상 5개 (각 10~15분)
├── 촬영: 탑뷰 + 정면 2앵글
├── 편집: CapCut or DaVinci Resolve (무료)
└── 보너스: 연습지 PDF 포함

판매 채널:
├── 웹사이트 직접 판매 (결제: 토스페이먼츠 연동)
├── 클래스101 입점 (수수료 30~40%, 대신 플랫폼 유입)
└── 크몽 (전자책/디지털 파일 카테고리)
```

**웹사이트 기술 구현 (이미 가능한 것들):**
- Supabase `products` 테이블 추가
- RLS로 구매자만 콘텐츠 접근 가능
- 결제: 토스페이먼츠 or 아임포트 연동 (Next.js API Routes 활용)
- `/digital-shop` 페이지 추가

## 전략 6: "기업/단체 출장 원데이 클래스" — 고단가 B2B

### 왜 이것인가
- 기업 워크숍/팀빌딩 시장에서 "캘리그라피 원데이"는 인기 프로그램
- 1회 출장으로 50~100만원 매출 가능 (수강료 대비 5~10배)
- 김포는 마곡, 여의도, 강서구와 가까워 서울 서부 기업 접근성 양호

### 구체적 첫 단계

**서비스 패키지 설계:**

| 패키지 | 인원 | 시간 | 내용 | 가격 |
|--------|------|------|------|------|
| 미니 | 5~10명 | 2시간 | 붓펜 캘리 기초 + 엽서 제작 | 50만원 |
| 스탠다드 | 10~20명 | 3시간 | 붓캘리 기초 + 족자/액자 작품 1점 | 80만원 |
| 프리미엄 | 20~30명 | 4시간 | 팀별 합작 작품 + 개인 작품 + 전시 | 120만원 |

**포함 사항:** 모든 재료, 작품 포장, 출장비 (김포 기준 30km 내 무료)

**첫 고객 확보 방법:**

1. **크몽 / 탈잉 등록** (즉시 실행 가능)
   - "기업 출장 캘리그라피 원데이 클래스" 서비스 등록
   - 가격: 50만원~
   - 포트폴리오: 기존 원데이 클래스 사진 활용

2. **김포 소재 기업 직접 컨택 (10곳)**
   ```
   타겟 기업 유형:
   ```

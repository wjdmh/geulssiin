### 3-A. 환경 분석 (External)
PESTEL 프레임워크 적용:

- Environmental: (해당 시 분석)
- Legal: 자격증/인증 관련 규제, 교육업 관련 법규

### 3-B. 산업 분석 (Industry)
Porter's Five Forces 적용:
- 기존 경쟁자 간 경쟁 강도
- 신규 진입자 위협
- 대체재 위협 (온라인 클래스, AI 캘리, DIY 키트 등)
- 공급자 교섭력
- 구매자(수강생) 교섭력

### 3-C. 내부 역량 분석 (Internal)
VRIO 프레임워크 적용:
- Value: 어떤 자원이 가치를 창출하는가?
- Rarity: 경쟁자가 쉽게 모방할 수 없는 자원은?
- Imitability: 복제 난이도가 높은 핵심 역량은?
- Organization: 이 자원들을 활용할 조직적 체계가 갖춰져 있는가?

### 3-D. SWOT 종합

- WO전략 (약점을 보완하여 기회 포착)
- ST전략 (강점으로 위협에 대응)
- WT전략 (약점과 위협을 최소화)

### 4-A. 수강생 확보 전략 (20명 → 30명+)

각 전략에 대해 아래 항목을 모두 포함:
1. 전략 설명 (무엇을, 왜)
2. 실행 방법 (내일부터 바로 실행 가능한 수준으로)
3. 필요 자원 (비용, 시간, 인력)
4. 예상 효과 (정량적 KPI 포함)
5. 리스크 및 대응책
6. 실행 타임라인 (주 단위)

분석해야 할 하위 주제:
- 신규 유입 채널 다변화 (온라인/오프라인)
- 원데이 클래스 → 정규반 전환율 극대화 전략
- 이탈 방지 및 수강 지속률 제고
- 입소문/추천 프로그램 설계
- 지역 타겟 마케팅 (김포 한강신도시 특성 고려)
- 연령대/관심사별 타겟 세그멘테이션
- 가격 전략 (현재 가격의 적정성, 패키지/멤버십 가능성)

### 4-B. 수익 다각화 전략

수강료 외 실행 가능한 모든 수익원을 분석하고, 각각에 대해:
1. BM(비즈니스 모델) 캔버스 요약
2. 예상 월 수익 (보수적/기본/낙관적 시나리오 3단계)
3. 초기 투자 비용 및 손익분기점까지의 기간
4. 기존 운영과의 시너지/충돌 분석
5. 실행 우선순위 (Impact/Effort 매트릭스)

고려할 수익원 후보 (이에 한정하지 말고 추가 제안도 해주세요):
- 실물 상품 판매 (도구 키트, 작품, 굿즈)
- 디지털 콘텐츠 판매 (PDF 가이드, 영상 레슨)
- B2B 서비스 (메뉴판, 간판, 기업 출장 클래스)
- 공간 임대/대여 (수업 외 시간활용)
- 자격증/수료증 프로그램의 프리미엄화
- 온라인 멤버십/구독 모델
- 작품 판매 및 커미션
- 제휴/콜라보 (지역 카페, 서점, 문화공간)
- 정부/지자체 보조금 및 지원사업 활용

### 4-C. 브랜드 & 마케팅 전략

1. **포지셔닝 전략:**
   - 현재 포지션 검증 및 최적 포지션 재설정
   - 경쟁자 대비 메시지 차별화
   - 타겟 고객 페르소나 3~5개 작성 (인구통계+행동+동기)

2. **채널 전략:**
   - 각 채널별(인스타/블로그/유튜브/네이버/카카오) ROI 분석
   - 채널별 콘텐츠 전략 및 게시 스케줄
   - SEO/지역검색 최적화 전략

3. **콘텐츠 마케팅:**
   - 콘텐츠 필러(Pillar) 3~5개 정의
   - 월간 콘텐츠 캘린더 템플릿
   - UGC(사용자 생성 콘텐츠) 활용 전략

4. **지역 마케팅:**
   - 김포 한강신도시 특성에 맞는 오프라인 마케팅
   - 지역 커뮤니티/맘카페 진입 전략
   - 제휴 파트너십 (카페, 서점, 문화공간, 학교)

### 4-D. 운영 최적화 전략

1. **시간 효율화:**
   - 현재 10개 슬롯 60% 가동률의 최적화 방안
   - 비수업 시간대의 수익화 전략
   - 대표 1인 운영의 시간 병목 분석 및 해결책

2. **가격 구조:**
   - 현재 가격 vs 시장 가격 벤치마킹
   - 가격 인상 가능성 및 방법론 (가치 기반 가격 책정)
   - 패키지/멤버십/장기 등록 할인 구조 설계

3. **고객 생애 가치(CLV) 극대화:**
   - 수강생 여정(Journey) 맵핑: 인지 → 체험 → 등록 → 지속 → 확장 → 추천
   - 각 단계별 전환율 개선 전략
   - 이탈 지점(Churn point) 식별 및 대응

### 4-E. 디지털 전환 전략

1. **웹사이트 활용 극대화:**
   - 현재 구축된 Next.js + Supabase 사이트의 전략적 활용 방안
   - 전환(Conversion) 최적화: 방문자 → 문의 → 수강생
   - 데이터 분석 프레임워크 (어떤 지표를 추적해야 하는지)

2. **온라인 수익 모델:**
   - 디지털 상품 판매 플랫폼 구축 전략
   - 온라인 클래스 런칭 시 자체 vs 플랫폼(클래스101, 탈잉) 비교 분석
   - 하이브리드(온+오프) 모델 설계

### 4-F. 리스크 관리 전략

1. **재무 리스크:**
   - 수강생 급감 시나리오별 대응 플랜 (15명, 10명, 5명 이하)
   - 고정비 구조 분석 및 최적화
   - 비상 자금 확보 방안

2. **운영 리스크:**
   - 대표 1인 의존 리스크 및 분산 방안
   - 시장 변화(AI, 온라인화) 대응 시나리오
   - 계절성/비수기 대응 전략

# 글씨인아트센터 — 웹페이지 구조 리뉴얼 아이디어

> 작성일: 2026-03-04
> 목적: 현재 웹사이트 구조를 분석하고, 브랜드 철학에 맞는 최적의 페이지 구조와 UX를 제안

---

## 1. 현재 사이트 구조 분석

### 현재 페이지 구성
```
/ (홈)          → Hero + 갤러리 프리뷰 + 시간표
/gallery        → 갤러리 (그리운 갤러리 / 회원 갤러리)
/about          → 센터 소개 + 대표 프로필
/class          → 수업 안내 (시간표)
/board          → 게시판
/login          → 로그인
/admin/*        → 관리자 페이지
```

### 현재 문제점
1. **메인 페이지가 단조로움** — Hero → 갤러리 → 시간표 3단 구조로, 센터의 철학이나 스토리가 전달되지 않음
2. **첫 방문자를 위한 동선 부재** — "이 센터가 뭐하는 곳이지?"에 대한 답이 즉시 보이지 않음
3. **수업 안내가 시간표뿐** — 각 수업의 특징, 커리큘럼, 분위기가 전달되지 않음
4. **모바일 내비게이션 미구현** — 모바일에서 메뉴 접근 불가
5. **CTA(행동 유도)가 약함** — "문의하기" 버튼이 네비게이션에만 존재

## 2. 제안하는 전체 사이트맵

```
/ (홈)
├── Hero (풀스크린)
├── 센터 한 줄 소개
├── 갤러리 하이라이트 (3~4점 큐레이션)
├── 수업 미리보기 (5개 클래스 카드)
├── 수강생 후기 / 커뮤니티 섹션
└── CTA (문의하기 / 체험 수업 신청)

/gallery
├── 그리운 갤러리 (대표 작가 작품)
├── 글씨인 회원 갤러리 (수강생 작품)
└── 작품 상세 + 구매 문의

/about
├── 브랜드 스토리 (스크롤 스토리텔링)
├── "글씨인" 이름의 의미
├── 그리운 작가 프로필
└── 센터 공간 소개

/class
├── 5개 수업 개별 섹션 (아코디언 or 탭)
│   ├── 수업 소개 / 특징
│   ├── 커리큘럼 단계
│   ├── 수업 사진
│   └── 수강료 / 일정
└── 원데이 클래스 특별 섹션

/board          → 커뮤니티 게시판
/login          → 로그인/회원가입
```

## 3. 메인 페이지 (/) 리뉴얼 상세

현재 메인은 **Hero + Gallery + Timetable** 3단 구조다. 이를 **6~7개 섹션의 스크롤 스토리텔링**으로 확장한다.

### 섹션 1: Hero (유지 + 강화)
```
현재: 배경 이미지 + "예술이 되는 글씨, 감성이 되는 그림" + "글씨인아트센터"
변경: 동일 구조 유지하되, 서브 카피를 브랜드 철학 문서의 카피로 교체

"잘 쓴 글씨가 아닌, 나다운 글씨를 찾아가는 시간"
또는
"예술이 되는 글씨, 감성이 되는 그림"  ← 현재 것도 충분히 좋음
```

**추가 아이디어:** Hero 하단에 부드러운 스크롤 화살표 + "붓끝에서 시작되는 당신의 이야기" 같은 마이크로 카피

### 섹션 2: 브랜드 인트로 (신규)
```
"디지털이 모든 것을 대체하는 시대,
손으로 쓴 글씨에는 대체할 수 없는 온기가 있습니다.
글씨인아트센터는 그 온기를 지키는 공간입니다."

[센터 내부 사진 or 먹물/붓 클로즈업 이미지]
```

- 배경: 부드러운 크림/아이보리 또는 화선지 텍스처
- 텍스트는 스크롤에 따라 한 줄씩 fade-in
- 이 섹션의 목적: 첫 방문자에게 "여기는 기존 학원과 다르다"는 인상을 즉시 전달

### 섹션 3: 갤러리 하이라이트 (현재 갤러리 프리뷰를 리디자인)
```
"작품으로 말하다"

[큐레이션된 3~4점의 대표 작품]
                                          → 갤러리 전체 보기
```

- 현재: 전체 갤러리를 메인에 노출 (과다)
- 변경: **3~4점만 큐레이션**하여 "맛보기"로 노출
- 관리자가 "메인 노출" 작품을 선택할 수 있도록 `is_featured` 플래그 추가 검토
- 레이아웃: 비대칭 2열 (큰 이미지 1 + 작은 이미지 2~3) — 매거진 느낌

### 섹션 4: 수업 미리보기 (신규)
```
"다섯 가지 길, 하나의 여정"

[붓 캘리그라피] [붓펜 캘리그라피] [펜드로잉] [서예] [원데이]
   전통의 깊이    일상의 실용     선으로 보는    전통의 뿌리   첫 만남
                                  세상
                                          → 수업 안내 전체 보기
```

- 각 카드: 대표 이미지 + 수업명 + 한 줄 설명
- 호버 시 카드 확대 + 간단한 커리큘럼 프리뷰
- 클릭 시 `/class` 페이지의 해당 수업 섹션으로 스크롤 이동

### 섹션 5: 수강생 후기 / 경험 (신규)
```
"글씨인의 이야기"

"처음엔 글씨를 배우러 왔는데,
 어느새 나를 만나는 시간이 되었어요."
                              — OO님, 붓 캘리그라피 수강 1년

[후기 카드 슬라이더 2~3개]
```

- 수강생 실제 후기 (관리자 입력 or 게시판 연동)
- 후기와 함께 해당 수강생의 작품 사진 1장 (선택)
- 이 섹션은 "사회적 증거(social proof)" — 방문자의 신뢰 확보

### 섹션 6: 그리운 작가 소개 (신규, 간결 버전)
```
"글씨인아트센터를 이끄는 사람"

[그리운 작가 사진]
그리운(號) — 글씨인아트센터 대표 작가
대한민국 미술대전(국전) 특선
KCDIA 정회원 및 인증강사

"글씨는 마음의 거울입니다."
                              → 더 알아보기 (About 페이지)
```

- About 페이지의 상세 프로필로 연결
- 메인에서는 핵심 경력 2~3줄 + 작가 코멘트만

### 섹션 7: CTA (행동 유도)
```
"첫 수업을 시작해보세요"

[전화 문의]  [카카오톡 문의]  [체험 수업 신청]
```

- 풀 와이드 배경 (짙은 색 or 먹물 텍스처)
- 세 개의 CTA 버튼: 전화, 카카오톡, 체험수업
- 체험수업 신청은 원데이 클래스 문의로 연결

## 4. 갤러리 페이지 (/gallery) 구조

### 현재 (구현 완료)
- "그리운 갤러리" / "글씨인 회원 갤러리" 탭 구분
- Masonry 레이아웃
- 작품 상세 모달 (이미지 + 정보 + 구매 문의)
- 판매 상태 배지 (판매중/판매완료)

### 추가 개선 아이디어

**A. 갤러리 상단 인트로**
```
그리운 갤러리

"그리운(號) — 그리움, 향수, 소중한 것에 대한 기억.
 글씨인아트센터 대표 작가의 작품 공간입니다."
```
- 탭 전환 시 해당 갤러리의 인트로 텍스트도 함께 변경

**B. 필터 / 정렬 기능 (향후)**
- 재료별 필터: 먹, 펜, 혼합
- 판매 상태 필터: 전체 / 판매중 / 판매완료
- 정렬: 최신순 / 오래된순

**C. "이달의 작품" 큐레이션 (향후)**
- 매월 1~2점을 "이달의 작품"으로 선정
- 갤러리 최상단에 크게 노출
- 작가의 작품 설명(스토리)을 함께 게시

## 5. About 페이지 (/about) 구조

### 현재 (구현 완료)
- 스크롤 스토리텔링 5섹션 구조
- 브랜드 철학 + 대표 프로필(타임라인) + 수업 프리뷰 + CTA

### 추가 개선 아이디어

**A. "글씨인" 이름의 의미 섹션**
```
글씨 + 인(人) → 글씨를 쓰는 사람
글씨 + 인(因) → 글씨로 인하여
글씨 + 인(印) → 글씨가 곧 도장
```
- 세 가지 의미를 시각적으로 표현 (한자 + 설명)
- 스크롤 시 하나씩 등장하는 애니메이션

**B. 센터 공간 소개**
- 센터 내부 사진 (수업 풍경, 도구, 작품 전시 공간)
- "오시는 길" 지도 (네이버 지도 or 카카오맵 임베드)
- 현재 Footer에만 있는 주소 정보를 여기서도 표시

**C. 성장 로드맵 시각화**
```
기초반 → 중급반 → 고급반 → 자격증반 → 작가반

"배우는 사람에서, 쓰는 사람으로.
 기초부터 작가까지, 오래 함께 걸어가는 길."
```
- 수평 타임라인 형태로 단계 표시
- 각 단계 클릭 시 해당 수업 설명 팝업

### 현재
- `TimetableSection` 하나로 모든 수업의 시간표만 표시

### 제안: 수업별 상세 섹션 + 시간표

**구조:**

## 7. 네비게이션 개선

### 현재 문제
- 모바일 메뉴 미구현 (`{/* Mobile menu implementation would go here */}`)
- 게시판 링크가 `navLinks` 배열 밖에서 하드코딩

### 제안

**A. 모바일 햄버거 메뉴 구현 (필수)**
- 햄버거 아이콘 → 풀스크린 오버레이 메뉴
- 메뉴 항목: 홈, 갤러리, 센터 소개, 수업 안내, 게시판, 문의하기
- AnimatePresence로 부드러운 열기/닫기

**B. 네비게이션 구조 정리**
```
[로고]  홈  갤러리  센터소개  수업안내  게시판  |  로그인  [문의하기]
```
- `navLinks` 배열에 게시판도 포함
- "문의하기" 버튼은 네비게이션 최우측 강조 버튼으로 유지

**C. 스크롤 기반 네비게이션 변화 (현재 구현됨, 유지)**
- 홈: 투명 → 스크롤 시 화이트
- 다른 페이지: 항상 화이트

## 8. 추가 기능 아이디어

### A. 원데이 클래스 예약 시스템 (중기)
```
/class/oneday
├── 이번 달 원데이 클래스 목록
│   ├── 날짜 / 시간
│   ├── 주제 (부채 만들기, 액자 캘리 등)
│   ├── 정원 / 잔여석
│   └── [예약하기] 버튼
└── 예약 폼 (이름, 연락처, 인원)
```

- Supabase에 `oneday_reservations` 테이블 추가
- 관리자가 원데이 클래스 일정 등록 → 웹에서 예약 가능
- 예약 시 카카오톡 알림 발송 (향후)

### B. 수강생 작품 업로드 기능 (중기)
- 로그인한 수강생이 직접 작품 업로드
- 관리자 승인 후 "글씨인 회원 갤러리"에 게시
- 수강생 프로필 페이지 (내 작품 모아보기)

### C. 블로그 / 소식 섹션 (장기)
```
/news
├── 센터 소식 (전시, 이벤트)
├── 수업 후기
└── 캘리그라피 팁 / 칼럼
```
- 현재 게시판을 확장하거나 별도 구성
- SEO에 유리한 콘텐츠 마케팅 채널

## 9. 디자인 시스템 제안

### 컬러 팔레트
```
Primary:    #1a1a1a (먹빛 블랙)
Secondary:  #f5f0eb (화선지 아이보리)
Accent:     #8b7355 (먹의 갈색 톤) 또는 #c4a882 (황토)
Text:       #333333 (본문), #666666 (보조), #999999 (캡션)
White:      #ffffff
```

### 타이포그래피 (현재 시스템 유지 + 강화)
```
제목 (Serif):  Noto Serif KR — 전통적 격조
본문 (Sans):   Noto Sans KR — 깔끔한 가독성
강조 카피:     Noto Serif KR Italic 또는 Light Weight
영문 보조:     현재 시스템 유지
```

### 여백과 리듬
- 섹션 간 간격: `py-24 md:py-32` (현재보다 더 넓게)
- 텍스트 블록 간: `space-y-6` ~ `space-y-8`
- "여백이 곧 브랜드"라는 원칙 — 빽빽하지 않고 숨 쉴 공간을 충분히

### 애니메이션 원칙
- **느리고 부드럽게:** `duration: 0.8~1.2s`, `ease: "easeOut"`
- **과하지 않게:** fade-in + 살짝 위로 올라오는 정도
- **의미 있게:** 스크롤에 따라 콘텐츠가 "드러나는" 느낌 (reveal, not bounce)

## 10. 구현 우선순위

### Phase 1 (즉시 — 1~2주)
1. **모바일 메뉴 구현** — 현재 모바일에서 네비게이션 불가, 치명적
2. **메인 페이지 브랜드 인트로 섹션 추가** — Hero 아래에 센터 한 줄 소개
3. **메인 갤러리 섹션을 하이라이트(큐레이션) 방식으로 변경**

### Phase 2 (단기 — 2~4주)
